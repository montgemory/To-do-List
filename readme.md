# TodoList 应用

这是一个基于 Vue.js 和 Node.js 的 TodoList 全栈应用。

## 项目结构

- `/frontend` - 前端应用 (Vue.js)
- `/backend` - 后端应用 (Node.js + PostgreSQL)

## 前端功能

1. 待办事项的增删改查
2. 待办事项的拖拽排序
3. 待办事项的筛选

技术要求：vue.js

UI/UX:
- Notion 风格的简约现代设计
- 全屏响应式布局
- 平滑的动画过渡效果
- 良好的用户体验

## 后端功能

1. 待办事项的增删改查
2. 待办事项的排序修改

技术栈：node.js 和 PostgreSQL 数据库

## 启动应用

### 后端

```bash
cd backend
npm install
npm start
```

### 前端

```bash
cd frontend
npm install
npm run dev
```
